二.分析及解决

  这个错误是由于 Java 9 引入的模块化系统所导致的安全限制。在 Java 9 之后的版本中，模块系统默认会限制对受保护或私有成员的访问。因此在idea中点击File，进入Project Structure，修改项目所使用的jdk版本为8。
